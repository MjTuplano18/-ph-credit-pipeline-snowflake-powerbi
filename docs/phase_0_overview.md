# Project Overview

See phase_all_walkthrough.md for the full end-to-end guide.

This project builds an ELT pipeline for a Philippine collections use case, using Lending Club data as a proxy and BSP macro indicators for context.

The goal is to identify which loan segments should be prioritized for collections given macroeconomic conditions.
